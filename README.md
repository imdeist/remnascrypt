# remnascrypt

Скрипты для быстрой установки SelfSNI, выпуска сертификатов Let's Encrypt и запуска `remnawave/node` на Debian/Ubuntu.

## Описание

Этот репозиторий предназначен для автоматической установки фейкового сайта, выпуска SSL-сертификатов и запуска ноды Remnawave на сервере. README обычно должен кратко объяснять, что делает проект, как его запустить и какие требования нужны для работы.[web:56][web:62]

Скрипт выполняет следующие действия:

- Проверяет запуск от `root`
- Проверяет, что система — Debian или Ubuntu
- Устанавливает зависимости
- Проверяет A-запись домена и внешний IP сервера
- Проверяет доступность портов `443` и `80`
- Выпускает сертификат Let's Encrypt
- Настраивает nginx под SelfSNI
- Создаёт конфигурацию для `remnawave/node`
- Запускает контейнер ноды

## Требования

Перед запуском убедитесь, что:

- У вас Debian или Ubuntu
- У вас есть домен с A-записью на IP сервера
- Порт `443` свободен
- Порт `80` свободен, если не используется `--without-80`
- Скрипт запускается от `root`
- У сервера есть доступ в интернет
